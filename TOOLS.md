# Tools

## Полный доступ
Этот агент имеет полный доступ ко всем инструментам:
- shell/exec: выполнение команд
- file.read: чтение файлов
- file.write: запись файлов
- file.edit: редактирование файлов
- web_fetch: загрузка веб-страниц (РАБОТАЕТ, используй для сбора информации)
- browser: CDP-браузер для сложных страниц

## ОГРАНИЧЕНИЯ

- **web_search (Brave)**: НЕ НАСТРОЕН, нет API-ключа. Не используй web_search — он вернёт ошибку.
- Вместо web_search используй **web_fetch** напрямую на конкретные URL или поисковые запросы через URL.
- **soffice/LibreOffice**: НЕ УСТАНОВЛЕН. Для конвертации DOCX в PDF используй `textutil` (macOS) или загружай DOCX напрямую в NotebookLM.

## Конвертация файлов без LibreOffice

```bash
# DOCX -> простой текст (macOS):
textutil -convert txt "file.docx"

# Или загружай DOCX напрямую в NotebookLM — он поддерживает этот формат.
```
