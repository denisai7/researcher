# Tools

## Полный доступ
Этот агент имеет полный доступ ко всем инструментам:
- shell: выполнение команд
- file.read: чтение файлов
- file.write: запись файлов
- file.edit: редактирование файлов
- web.search: поиск в интернете
- web.fetch: загрузка веб-страниц

Никакие инструменты не заблокированы.
